# Meta Ads Intelligence Report

An interactive, high-end, and visually stunning web-based dashboard designed for Meta Ads Audits and Competitor Intelligence. This fully responsive HTML/CSS/JS single-page report delivers deep marketing insights with a dark-mode cyberpunk aesthetic, dynamic components, and clean UI/UX.

## Features & Architecture

The report is systematically structured into key intelligence sections to ensure a seamless client presentation:

- Performance Overview (`#kpis`): High-level core metrics and target KPIs with animated counter components.
- Live Ad Inventory (`#ads`): Showcase of current active ad accounts, live statuses, and structural health.
- Creative Performance (`#creatives`): Multi-angle visual analysis of active ad formats, hooks, and copies.
- Competitor Ads Analysis (`#competitors`): Deconstructed strategies, budget positioning, and real market telemetry.
- Advanced Intelligence (`#intel`): Data science and audience insights utilizing custom CSS-rendered performance tracking charts (e.g., ROAS charts).
- Market Opportunities (`#opportunities`): Visualized blue-ocean growth tactics and scalability frameworks.
- Priority Recommendations (`#recommendations`): Bulletproof, categorized next steps with actionable priority tagging.

## Tech Stack & Highlights

- Pure HTML5 & Semantic Sections: Organized flawlessly for structural clarity.
- Premium CSS3 Grid & Styling: Styled using the ultra-modern Inter and JetBrains Mono typography.
- Tailored Palette: Built over a premium `#060608` deep dark palette with electric purple (`#7c3aed`), royal blue, and cyan neon accents.
- Zero Heavy Dependencies: Light, super fast, and completely self-contained.
- Interactive JavaScript Utilities:
  - Dynamic Counters: Smooth number counter animation powered by IntersectionObserver.
  - Animated Charts: Pure CSS bar charts that animate gracefully upon scrolling into view.
  - Sticky Navigation Link-Highlighter: Tracks active user viewport and dynamically highlights current section index.
